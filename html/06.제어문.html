6장. 제어문

제어문이란?

제어문은 조건에 따라 코드 블록을 실행하거나(조건문) 반복 실행할 때(반복문) 사용한다. 일반적으로 코드는 위에서 아래 방향으로 순차적으로 실행된다. 제어문을 사용하면 코드의 실행 흐름을 인위적으로 제어할 수 있따.

블록문이란?

0개이상의 문능ㄹ 중괄호로 묶은 것으로 , 코드 블록 또는 블록이라고 칭하고 자바스크립트는 블목문을 하나의 실행 단위로 취급한다.

조건문인란?

주어진 조건식의 평가 결과, 논리적 참 또는 거짓에 따라 실행할 코드를 결정한다.

if문

if(조건식1){ 조건식1이 참일 경우 }else if(조건식2){ 조건식2이 참일 경우 }else{ 조건식들이 모두 거짓일 경우 실행 }

switch문

switch문은 주어진 표현식을 평가하여 그 값과 일치하는 표현식을 갖는 case 문으로 실행 흐름을 가진다.

switch문은 표현식과 일치하는 문이 없이 없다면 default문으로 이동한다.

switch(표현식){ case 표현식1: switch 문의 표현식1이 일치하면 실행될 문; break; case 표현식2 switch 문의 표현식1이 일치하면 실행될 문; break; default: switch 문의 표현식과 일치하는 case문이 없을 때 실행될 문; }

여기서 주의점은 case에 break문을 사용하지 않으면 맨 마지막 default의 문이 실행이 되는데 그 이유는 계속해서 case 값이 재할당이 되어 마지막 default 값이 재할당이 된다. 이를 폴스루라 한다. 그래서 break문을 사용하면 case를 빠저 나오도록 하자

반복문

반복문은 조건식의 평과 결과가 참인 경우 코드 블록을 실행한다. 그 후 조건식을 다시 평가하여 여전히 참인 경우 코드 블록을 다시 실행한다. 이는 조건식이 거짓일 때 까지 반복한다.

for문

for(var i = 0; i < 2; i++){ console.log(i); } 1. for문을 실행하면 var i = 0;을 선언 할당한 것이 한번 실행 2. 선언 할당한 것이 i < 2가 참인지 판별 3. 참이면 console.log(i); 실행 4. i++ 실행 5. 다시 i < 2 가 참인지 확인 6. 참일 경우 console.log(i) 실행 7. 코드 블록이 거짓일 경우 반복문 종료

for(var i = 0; i < 2; i++){ for(var j = 0; j < 1; j++){ console.log(i+j); } } 이중으로도 가능 이중 포문

while문

while문은 조건식의 평가 결과가 참이면 코드 블록을 계속해서 반복 실행한다. for문과 다른 점은 for문은 반복 횟수가 명확할 때 쓰고 while문은 반복 횟수가 불명확 할 때 사용, 조건식의 결과 불리언 값이 아니면 강제로 불리언으로 변환한다. 조건식의 평가가 언제나 참이면 무한루푸가 되고 if문에 break문으로 탈출을 한다.

do...while문

do...while문은 코드 블록을 먼저 실행을 하고 조건식을 평가한다. 따라서 코드 블록은 무조건 한번이상 실행이 된다.

var count = 0; do{ console.log(count);/0 1 2 count++; }while(count < 3)

break문

breake문은 코드 불록을 탈출할 떄 사용한다.

for문, for...in, for...of, while, do...while, switch 외에 사용하면 문법에러가 발생한다

continue문

continue문은 반목문의 코드 블록 실행을 현 지점에서 중단하고 반복문의 증감식으로 실행 흐름을 이동시킨다. break문 처럼 반복물을 탈출하지 않는다.

var string = 'hello world'; var search = 'l'; var count = 0; for(var i = 0; i < string.length; i++){ if(string[i] !== search) continue; count++; } console.log(count); //3
