5장. 연산자

연산자란?

하나 이상의 표현식을 대상으로 산술, 할당, 비교, 논리, 타입, 지수 연산들을 수행하는 하나의 값을 만든다.

연산자: +, -, *, /, %, = , > , &&, typeOf

피연산자: 값

산술 연산자

산술 연산자란?

피연산자를 대상으로 수학적인 계산을 수행해 새로운 숫자 값을 만든다. 불가능할 경우 NaN을 변환

이항 산술 연산자

2개의 피연사자를 산술 연산하여 숫자 값 만든다.

덧셈: 2+2=4

뺄셈: 2-2=0

곱셉: 2*5=10

나누셈: 10/2=5

나머지: 10%3=1

단항산술 연산자

1개의 피연산자를 산술 연산하여 숫자 값을 만든다.

var x = 5

선할당 후 증가

resulut = x++

console.log(resulut,x) //5,6

선증가 후할당

resulut = ++x

console.log(resulut,x) //7,7

선할당 후감소

resulut = x--

console.log(resulut,x) //7,6

선감소 후할당

resulut = --x

console.log(resulut,x) //5,5

- 값 변환

-'10' === -10

-true === -1

문자열 연결 연산자

+ 연산자는 피연산자중 하나 이사의 문자열인 경우 문자열 연결 연산자로 동작을 한다.

'1'+2=12

1+'2'=12

1+2=3

1+true=2 //true는 1로 타입 변환된다.

1+false=1// false는 0으로 변환된다.

1+null=1 // null은 0으로 타입 변환된다.

undefined는 타입 변환 불가

true,false,null 처럼 숫자타입으로 변환이 되는 것을 암묵적 타입 변환 또는 타입 강제 변환이라고 한다.

할당 연산자

우항에 있는 피연산자의 평과 결과를 좌항에 있는 변수에 할당한다

x=5

x+=5 // x= x+5

x-=5 // x= x-5

x*=5 // x= x*5

x/=5 // x= x/5

x%=5 // x= x%5

할당문은 표현식, 할당문은 값으로 평가되는 표현식인 문으로서 할당된 값으로 평가된다.

비교 연산자

비교 연산자는 좌항과 우항의 피연산자를 비교한 다음 그 결과를 논리형으로 변환을 한다.

== 동등 비교 //값 비교

=== 일치 비교 //타입까지 비교

!= 부동등 비교 // x와 y 값이 다를 경우 true

!== 불일치 비교 // x와 y 값이 타입까지 다를 경우 false

동등 비교(==) 연산자는 좌항과 우항의 피연산자를 비교할 때 먼저 암묵적 타입 변환을 통해 타입을 일치시킨 후 같은 값인지 비교한다.

일치 비교(===) 연산자는 좌항과 우항의 피연산자가 타입도 같고 값도 같은 경우에 한하여 true를 반환한다.

삼항 조건 연산자

조건식의 평가에 따라 반환할 값을 결정한다.

var resule = score >= 60 ? "pass" : "fail"

참일 경우 pass

거짓일 경우 fail

중요 삼항 조건 연산자 표현식은 값으로 평가할수 있는 표현식이다 문이 아니다
