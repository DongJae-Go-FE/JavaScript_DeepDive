24.dom

DOM(Document Objec Model)은 HTML 문서 계층적 구조와 정보를 표현하며 이를 제얼할 수 있는 API, 즉 프로퍼티와 메서드를 제공하는 트리 자료 구조다

노드란?

HTML DOM은 노드라고 불리는 계층적 단위에 정보 저장

노드 종류

  • 문서노드: 문서 전체를 나타내는 노드
  • 요소노드 : HTML 요소는 요소노드, 속성노드를 가질 수 있음
  • 속성(어트리뷰트)노드 : 속성은 모두 속성 노드이며, 요소 노드에 관한 정보를 가지고 있음
  • 텍스트노드: 텍스트는 모두 텍스트노드
  • 주석노드: 주석은 모두 주석 노드

노드 객체들로 구성된 츠티 자료구조를 DOM이라한다. 그래서 DOM을 DOM트리라고 부르기도 한다.

DOM은 HTML 문서의 계층적 구조와 정보를 표현하는 것은 물론 노드의 객체의 정류, 즉 노드 타입에 따라 필요한 기능을 프로퍼티와 메서드의 집합인 DOM API로 제공한다. 이 DOM API를 통해 html의 구조나 내용 또는 스타일 등을 동적으로 조작할 수 있다.
