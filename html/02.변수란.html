2장. 변수란?

변수는 하나의 값을 저장하기 위해 확보한 메모리 공간 자체 또는 그 메모리를 공간을 식별하기 위해 붙인 이름을 말한다.

즉 값의 위치를 가르키는 상징적인 이름

변수명: 메모리 공간에 저장된 값을 식별할 수 있는 고유한 이름

변수값: 변수에 저장된 값

할당: 변수에 값을 저장하는 것

변수명 === 식별자는 어떤 값을 구별해서 식별할 수 있는 고유한 이름

식별자는 값이 아니라 고유한 메모리 주소를 기억하고 있다.

변수를 사용하려면 반드시 선언이 필요하다 var, let, const

undefined는 자바스크립트에서 제공하는 원시 타입 값

호이스팅: 변수 선언이 소스코드가 한 줄씩 순차적으로 실행되는 시점, 즉 런타임이 아니라 그 이전 단계에서 먼저 실행된다. 즉 최상단에서 실행

변수 선언문이 코드의 선두로 끌어 올려진 것처럼 동작하는 자바스크립트의 고유한 특징을 변수 호이스팅이라고 한다.

변수 선언은 소스코드가 순차적으로 실행되는 시점인 런타임 이전에 먼저 실행되지만 값의 할당은 소스코드가 순차적으로 실행되는 런타임 시점에 실행된다.

값을 재할당할 수 없어서 변수에 저장된 값을 변경할 수 없다면 변수가 아니라 상수라 한다. const
