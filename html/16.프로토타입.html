16장.프로토타입

자바스크립트는 객체 기반의 프로그래밍 언어이며 자바스크립트를 이루고 있는 모든 것은 객체이다.

객체지향 프로그래밍

객체지향 프로그래밍은 객체의 집합으로 프로그램을 표현하려는 프로그래밍 패러다임을 말한다.

객체지향 프로그래밍의 특징

  • 실체는 특징이나 성질을 나타내는 속성을 가지고 있다.
  • 프로그램에 필요한 속성만을 간추려 표현하는 것을 추상화

위에 처럼 속성을 통해 여러 개의 값을 하나의 단위로 구성한 복합적인 자료구조를 객체라한다.

위에 처럼 객체지향 프로그래밍은 객체의 상태를 나타내는 데이터와 상태 데이터를 조작할 수 있는 동작을 하나의 논리적인 단위로 묶어 생각한다.

따라서 객체는 상태 데이터와 동작을 하나의 논리적인 단위로 묶은 복합적인 자료구조이다.

자바스크립트는 동일한 생성자 함수에 의해 생성된 모든 인스턴스가 동일한 매서드를 중복 소유하는 것은 메모리를 낭비하는 것과 같다

자바스크립트는 프로토타입을 기반으로 상속을 구현한다.

모든 객체는 __proto__ 접근자 프로퍼티를 통해 자신의 프로토타입, 즉[[Prototype]] 내부 슬롯에 간접적으로 접근할 수 있다.
